This repository contains modular, beginner-friendly code examples demonstrating the core components of LangChain. It was built as a companion to my deep technical blog post.

## 🚀 Getting Started

   ```bash
   pip install -r requirements.txt


📁 Repository Structure
01_basic_chain.py: Demonstrates LLM initialization, Prompt Templates, and LCEL (LangChain Expression Language).

02_memory_chatbot.py: Shows how to maintain state and context using ConversationBufferMemory.

03_agent_with_tools.py: A reasoning engine that autonomously uses the Wikipedia API to answer complex queries.

🧠 Concepts Covered
Orchestration: Chaining components together.

Statefulness: Injecting memory into stateless LLMs.

Tool Integration: Connecting AI to external knowledge bases.
